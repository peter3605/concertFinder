This is the placeholder index.html. The real React app is populated into internal/http/spa/static/ by the Docker build (see Dockerfile) before the Go binary is compiled.

For local development, run npm --prefix web run dev and hit https://127.0.0.1:3000 — Vite serves the real SPA and proxies /api/* to this Go server.

To rebuild the embedded SPA locally without Docker:
npm --prefix web run build && rm -rf internal/http/spa/static/* && cp -r web/dist/* internal/http/spa/static/
